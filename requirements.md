# Требования к информационной системе
«Система управления библиотекой»

1. Назначение системы
Информационная система «Управление библиотекой» предназначена для автоматизации
основных процессов работы библиотеки, включая учёт книг, регистрацию пользователей,
выдачу и возврат литературы, а также поиск информации о книгах.
2. Пользователи системы
В системе предусмотрены следующие категории пользователей:
- Читатель
- Библиотекарь
- Администратор
3. Функциональные требования
3.1 Функции читателя
- Поиск книг по названию, автору или жанру
- Просмотр информации о книге
- Просмотр статуса доступности книги
- Просмотр списка выданных книг
3.2 Функции библиотекаря
- Регистрация новых читателей
- Выдача книг читателям
- Возврат книг в библиотеку
- Учёт фонда библиотеки
- Обновление информации о книгах
3.3 Функции администратора
- Управление пользователями системы
- Настройка прав доступа
- Контроль корректности работы системы
4. Нефункциональные требования
- Удобный и понятный пользовательский интерфейс
- Надёжное хранение данных
- Доступ к системе через веб-браузер
